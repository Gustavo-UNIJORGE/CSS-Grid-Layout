Automatizando o Grid Layout

Grid Row

Com a propriedade grid-row definimos a linha de inicio e de final do item. É uma abreviação do grid-row-start + grid-row-end.

A propriedade grid-auto-columns e grid-auto-columns também são conhecidas como Implicity Grid Tracks (Trilhas de Grade Implicitas) uma vez que é possível defini-las arbitrariamente sem o propriedade inicial de grid-template-column ou grid-template-row.

.item-a {
grid-row: 1 / 3;
}
.item-b {
grid-row: 2;
}
