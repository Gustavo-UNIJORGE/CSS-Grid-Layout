Estilizando o Grid

Row Gap

A propriedade row-gap serve para definir o espaçamento entre as linhas presente no container, podendo ser um valor fixo ou flexível.

Essa propriedade também é conhecida como grid-row-gap mas provavelmente essa forma antiga de chamar a propriedade logo deve cair em desuso.

.grid {
row-gap: 1em;
grid-row-gap: 1em;
}
