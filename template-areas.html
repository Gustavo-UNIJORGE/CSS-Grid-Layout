Usando CSS Grid

Grid Template Areas

A propriedade grid-template-aras define um template de grade usando referencias nominais (de nome) para a disposição de areas nas linhas e colunas definidas anteriormente.

Os itens são atribuídos pelos nomes da grid area.

.item-a {
grid-area: header;
}
.item-b {
grid-area: main;
}
.item-c {
grid-area: sidebar;
}
item-d {
grid-area: footer;
}
.container {
display: grid;
grid-template-columns: 50px 50px 50px 50px;
grid-template-rows: auto;
grid-template-areas:
"header header header header"
"main main . sidebar"
"footer footer footer footer";
}
