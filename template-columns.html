Estruturando o Grid Layout

Grid Template Columns

A propriedade grid-template-columns serve para definir o número de colunas presente no container, podendo ser um valor fixo

  • px(pixels),
  • in(polegadas),
  • cm(centímetros)

ou um valor relativo como :

  • em(relativo ao font-size)
  • %(porcentagem)
  • vw(largura do viewport)
  • vh(altura do viewport)
  • ch(largura do "0" na fonte atual)
  • ex(altura do "x" na fonte atual)

e definindo a quantidade de colunas pela repetição dos seus valores, podendo também ser usando a notação repeat()

.grid {
display: grid;
grid-template-columns: 1fr 1fr 1fr;
}
Observe que nesse exemplo usamos a medida fr que é uma unidade relativa exclusiva do Grid Layout, servindo para distribuir igualmente os itens no espaço disponível.
