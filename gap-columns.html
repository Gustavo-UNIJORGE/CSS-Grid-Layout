Estruturando o Grid Layout

É possíve definir os intervalos entre as celulas do Grid usando propriedades Gap, seja column-gap, para definir o espaçamento entre as colunas, ou row-gap, para o espaçamento entre as linhas.

Column Gap

A propriedade column-gap serve para definir o espaçamento entre as colunas presente no container, podendo ser um valor fixo ou flexível.

Essa propriedade também é conhecida como grid-column-gap mas provavelmente essa forma antiga de chamar a propriedade logo deve cair em desuso.

.grid {
column-gap: 1em;
grid-column-gap: 1em;
}
