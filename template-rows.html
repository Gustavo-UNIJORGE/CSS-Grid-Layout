Usando CSS Grid

Grid Template Rows

Semelhante a propriedade grid-template-columns, a grid-template-rows serva para definir a estrutura do container a partir do número de linhas sendo um valor fixo ou flexível.

.grid {
display: grid;
grid-template-columns: 1fr 1fr;
grid-template-rows: 1fr 1fr 1fr;
}
